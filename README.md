# 🌐 Sanjay Portfolio

A modern, responsive developer portfolio showcasing my projects, technical skills, education, and experience. Built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui to provide a fast and engaging user experience.

---

* Modern responsive design
* Interactive landing page
* About Me section

* Featured Projects
* Education & Certifications

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/SANJAYGANDHI67/Portfolio.git
```

### Navigate to the project

```bash
cd Portfolio
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open the local URL shown in the terminal (typically `http://localhost:5173`).
